# 数据与基准 — 评测基石

模型和数据是硬币的两面。本板块收录高质量病理数据集及标准化基准测试，帮助用户验证模型泛化能力、公平对比算法性能。

## 收录标准

- 具有明确学术引用
- 已被领域广泛使用或为最新权威来源
- 明确标注获取方式和使用许可

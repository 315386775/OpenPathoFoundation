# 模型ZOO — 选型指南

收录已开源的病理基础模型与专用模型，通过可筛选/可对比的总表，帮助用户快速锁定候选模型。

## 模型总表

| 模型名称 | 发布机构 | 核心任务 | 参数量 | 训练数据规模 | 算法架构 | 开源情况 | 推荐场景 |
|---------|---------|---------|--------|------------|---------|---------|---------|

## 收录范围

目前开源病理基础模型生态已包括：

- **UNI** — Mahmood Lab 发布的通用病理视觉特征提取模型，基于 DINOv2 架构，在 100K+ WSI 上预训练
- **Virchow** — 大规模病理基础模型，基于 ViT 架构，近 150 万张 WSI 训练
- **Prov-GigaPath** — 基于 GigaPath 的全切片级病理基础模型
- **Phikon** — 基于 iBOT 的病理视觉特征学习
- **PLIP** — 病理视觉-语言预训练模型
- **CONCH** — 视觉-语言多模态病理模型，基于 CLIP 架构
- **GPFM** — 港科大发布的统一知识蒸馏预训练框架，效果碾压 UNI、CHIEF
- **CHIEF** — 病理图像特征提取基础模型

此外，达摩院的 **Lingshu** 等医学多模态大模型也值得关注。

## 如何选择模型

- **通用下游任务微调** → UNI、CHIEF、Phikon
- **大参数量、强泛化能力** → Virchow、Prov-GigaPath
- **视觉-语言多模态推理** → CONCH、PLIP
- **中国人群/临床诊断场景** → RuiPath
